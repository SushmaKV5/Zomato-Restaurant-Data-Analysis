# Zomato Restaurant Data Analysis

## 📌 Objective
To analyze restaurant data and identify trends in ratings, pricing, cuisines, and locations.

## 🔧 Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## 📊 Key Insights
- Majority of top-rated restaurants are concentrated in metro areas
- Certain cuisines dominate high-rating segments
- Moderate pricing correlates with higher ratings
